# 项目理解
## 主控部分选用ESP32因其自带WIFI连接功能用于确定时间和校准，相较于STM32更加简洁
## 灯板材料用WS2812C，因其亮度高，采用5V供电，可直接连接到ESP32系统板的5V端口
## 灯板数量为六块即时针分针秒针各两块
## 考虑到电池体积及ESP32的3.3V端口，电池最终选用3.7v供电
